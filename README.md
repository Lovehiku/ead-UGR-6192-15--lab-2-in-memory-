# PizzaStore In-Memory API

This is a simple Minimal API using **EF Core In-Memory Database** for storing pizzas.

## Features
- List all pizzas (`GET /pizzas`)
- Get a single pizza (`GET /pizza/{id}`)
- Add a new pizza (`POST /pizza`)
- Update a pizza (`PUT /pizza/{id}`)
- Delete a pizza (`DELETE /pizza/{id}`)
- Swagger UI available at `/swagger`
